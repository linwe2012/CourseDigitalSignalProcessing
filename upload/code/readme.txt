This folder contains:

Script main.m -- merge different images to gain an artistic style.
Folder artistic -- test images for main.m

Script main_hdr.m -- implement hdr
    - readImages.m -- read in images.
    - alignImages.m -- optional function to align images.
    - imagesToMatrix.m -- sample images and fit it to matrix.
    - gsolve.m -- solve the g.(See Report for details)
    - hdrMap.m -- generate hdr map
    - colorSampling.m -- reconstruct color.
Folder img -- test images
   - results -- the result generated from different configs.
                     The config is specified by the filename.
